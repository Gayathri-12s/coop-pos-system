# COOP Supermarket POS System

A full-stack Point of Sale system built using:

- React.js
- Django REST Framework
- MySQL
- Bootstrap 5
- JWT Authentication

## Features

- Secure Login
- Product Search
- Category Filters
- Shopping Cart
- VAT Calculation
- Receipt Generation
- Sales Tracking
- Inventory Management

## Tech Stack

Frontend:
- React
- Bootstrap 5

Backend:
- Django
- Django REST Framework

Database:
- MySQL
